Abbiamo deciso di ideare un nostro esercizio che vuole rappresentare una pagina web 
esplicativa sui Linguaggi HTML e CSS.
La pagina è composta da una pagina principale (file index.html) con riferimenti ipertestuali 
ad altre pagine HTML. Sono state create 4 pagine in totale :
	1 ) Una pagina index.html per svolgere il ruolo di "home"
	2 ) Una pagina intro.html per fornire una breve introduzione sui due linguaggi
	3 ) Una pagina storia.html per fornire una prospettiva storica dei due linguaggi
	4 ) Una pagina funzionalita.html per mostrare le potenzialità dei due linguaggi

Abbiamo cercato di affrontare ogni aspetto dei due linguaggi anche inserendo 
particolarità che abbiamo approfondito personalmente (specialmente con CSS)

Per HTML abbiamo cercato di utilizzare la maggior parte degli elementi possibili

Con CSS oltre alle proprietà viste a lezione abbiamo anche aggiunto delle 
particolarità come ad esempio l'introduzione di una durata per l'animazione
