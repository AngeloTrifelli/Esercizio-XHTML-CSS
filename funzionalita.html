Le funzionalità di HTML e CSS

HTML attraverso l'uso di tag e degli elementi permette di poter creare lo scheletro della nostra pagina web e non solo...

Ora elencheremo alcune delle possibili opportunità che HTML mette a disposizione:
Analizziamo i primi semplici elementi con relativi tag

  • <h1> elemento block-level utilizzato per il titolo principale(utilizzando h2,h3 fino ad h6 avremo titoli che avranno un importanza gerarchica sempre minore)
  • <p> è l'elemento utilizzato per creare paragrafi
  • <a> è l'elemento utilizzato per link esterni
  • <em> è l'elemento inline per testo in corsivo
  • <br> è l'elemento non strutturle utilizzato per andare a capo
  • <strong> è l'elemento inline per il grassetto
  • <ul> è l'elemento lista per creare liste non ordinate(lista che contiene elementi 'punto di lista')
  • <il> è l'elemento punto di lista
  • <img> è l'elemento utilizzato per inserire un'immagine

In particolare, con HTML gli elementi vengono distinti in due categorie :


  1. Elementi block-level: vengono visualizzati per default nettamente separati dagli altri elementi. Ad esempio <p> e <table> fanno parte di questa categoria.
  2. Elementi in-line: vengono visualizzati con continuità rispetto al testo che li circonda. Ad esempio <img> , <strong> <a> , <em> , ... fanno parte di questa categoria

Abbiamo visto come con HTML possiamo costruire le fondamenta della nostra pagina web.
Ma a livello estetico il vero lavoro lo fa il CSS. Con il CSS possiamo decidere il layout della nostra pagina, come vogliamo che venga visualizzato, che colore devono essere le scritte o il nostro sfondo. Ad esempio la tabella precedente attraerso il CSS può assumere queste sembianze:

Ma questo è solo un banalissimo esempio. L'esempio più adatto è la pagina stessa che ora state visualizzando. L'intero layout è stato realizzato attraverso il CSS!

In particolare, con CSS le regole possono essere definite in tre modi distinti:


  1. Regole inline: definite direttamente nell'elemento che governano, mediante l'attributo: "style"
  2. Internal stylesheet: definite nella <head> del documento, mediante il tag: <style>
  3. External stylesheet: definite in un file a parte che poi viene riferito con un elemento <link> del documento


Per definire una regola si associa:


  • Un selettore: elemento o "tipi di elemento" a cui applicare la regola.
  • Insieme di comandi di stile: racchiusi tra {...}

selettore {property: value; ... ; property: value}
